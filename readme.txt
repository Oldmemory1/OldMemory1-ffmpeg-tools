注意，使用该软件包前，你需要安装ffmpeg，并且将其添加到环境变量中，建议Windows环境下使用scoop包管理器安装ffmpeg，Linux或MacOS环境使用apt-get、pacman或brew等包管理器安装ffmpeg。
convert_m4a_to_flac.py：
（1）将一个目录下所有的m4a文件转换为flac格式；
（2）将某个指定路径的m4a文件转换为flac格式；
convert_mp3_to_flac.py：
（1）将一个目录下所有的mp3文件转换为flac格式；
（2）将某个指定路径的mp3文件转换为flac格式；
convert_wav_to_flac.py：
（1）将一个目录下所有的wav文件转换为flac格式；
（2）将某个指定路径的wav文件转换为flac格式；
merge_equal_long_m4a_and_mp4.py：
（1）将mp4视频和m4a音轨合并为新的mp4文件
（2）批量合并某个指定目录下所有mp4文件和其同名的m4a音轨文件为新的mp4文件
extract_m4a_audio_track_from_mp4.py：
（1）从mp4文件中提取完整的m4a音轨
（2）批量提取某个指定目录下所有mp4文件中完整的的m4a音轨
